# 🍔 TenderBite – Smart Food Discovery App

## 🚀 Overview
TenderBite is a swipe-based food discovery app designed to eliminate the everyday problem of deciding what to eat.

Inspired by the simplicity of swipe interactions, TenderBite allows users to quickly browse food options, match preferences, and make decisions individually or as a group.

---

## ✨ Core Concept
Instead of endlessly scrolling through food delivery apps, users can:
- Swipe right on dishes they like  
- Swipe left on dishes they dislike  
- Instantly narrow down choices  

TenderBite transforms food selection into a fast, engaging, and intuitive experience.

---

## 🔥 Key Features

### 🍽️ Swipe-Based Discovery
- Tinder-style interface for food selection  
- Smooth and intuitive user interactions  
- Reduces decision fatigue  

### 👥 Group Decision System
- Create or join groups using a unique code  
- Each user swipes individually  
- System generates the best common matches  

### 🧠 Smart Recommendations
- Personalized suggestions based on preferences  
- Optimized results for faster decision-making  

### 💸 Price Comparison (Planned)
- Compare food prices across delivery platforms  
- Find the best deals instantly
